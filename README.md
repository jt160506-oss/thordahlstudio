# Thordahl — thordahlstudio.dk

Premium digitalt webdesign-studie (personligt brand: Jakob Thordahl) for små danske virksomheder.
Niche: hjemmesider bygget til at blive fundet på Google **og** anbefalet af AI (ChatGPT/Perplexity).

Statisk site — HTML/CSS/vanilla JS. Hostet gratis på GitHub Pages. Domæne: thordahlstudio.dk.

## Filer
- `index.html` — forside (B×C hero-prototype, optimeret med SEO + tilgængelighed)
- `404.html` — fejlside
- `assets/` — favicon, og-image og øvrige aktiver
- `robots.txt`, `sitemap.xml`, `llms.txt` — synlighed for søgemaskiner og AI
- `CNAME` — kobler GitHub Pages til thordahlstudio.dk
- `AGENTS.md` — fuld arbejdsguide (retning, sandhedsregler, QA-tjekliste)
- `PROJEKT-INSTRUKTION.md` — kort version
- `reference/` — skærmbilleder af den valgte retning

## Arbejdsmåde
Arbejd altid på en branch, aldrig direkte på `main`. Preview via staging-URL før live.
